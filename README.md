While Loop: Repeatedly executes a block of code as long as a specified condition remains true. It is typically used when the number of iterations is not known in advance.
For Loop: Used for iterating over a sequence (like a list, tuple, or range). It is ideal for situations where you know exactly how many times the code should run.
Break: Immediately terminates the loop.
Continue: Skips the current iteration and moves to the next one.
Nested Loops: A loop inside another loop, essential for working with multi-dimensional structures like matrices.
